About the Speedtest

The Speedtest 2026 is a challenge for vocational web development students. You have one hour to complete ten different tasks. You cannot use AI tools or the internet during the test.

The first 5 tasks

Task 1: CSS tabbed content

Only use CSS to toggle the content of corresponding tabs.

Task 2: Make a flag

Recreate the swiss flag.

Task 3: Explain your choices

Write short comments in your code to explain what you changed and why.

Task 4: JavaScript arrays

Time to put your array-skills to the test.

Task 5: HTML elements

Improve this website by adding the correct HTML elements as described in the .md file.

Test information

  • Duration: 60 minutes
  • Number of tasks: 10
  • Tools allowed: Local editor and browser only
    • Start practice

    Tip: Focus on tasks you can finish quickly. Some tasks are harder than others. All result in equal points.
